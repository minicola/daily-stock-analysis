# A股市场推荐技能

## 目标
根据交易时段分析市场，**结合日K线和成交额的五维技术分析**筛选并推荐3只股票，提供趋势判断和操作建议。

## 核心原则

- **数据驱动**：每只推荐股票必须基于30日日K线数据进行技术分析，不得仅凭当日涨幅推荐
- **趋势验证**：通过均线系统、量价关系、动量指标三重验证趋势方向
- **评分筛选**：候选股必须通过简化五维评分（≥60分方可推荐），低于60分不推荐
- **概率表达**：使用概率化语言描述趋势（如"上涨概率70%"），不做绝对预判
- **成本真实**：所有买卖建议的资金计算必须包含交易手续费，不得只算股价×数量

## 候选股排除规则（硬性）

在筛选候选股时，**必须首先排除以下类型**，排除后再进行五维评分：

| 排除类型 | 识别规则 | 原因 |
|---------|---------|------|
| **创业板** | 股票代码以 `300xxx` 或 `301xxx` 开头 | 开通门槛高、波动大、20%涨跌幅，不适合普通投资者 |
| **ST/\*ST** | 股票名称包含 `ST`、`*ST`、`S` 前缀 | 退市风险、基本面恶化、5%涨跌幅限制 |

**判定方式**：
```python
def is_excluded(code: str, name: str) -> bool:
    """排除创业板和ST股票"""
    # 创业板: 300xxx, 301xxx
    if code.startswith('300') or code.startswith('301'):
        return True
    # ST股票: 名称含ST
    if 'ST' in name.upper():
        return True
    return False
```

**注意**：
- 即使创业板/ST股票当日涨幅领先或板块热度高，也**一律不推荐**
- 如果用户已持有创业板/ST股票，在持仓操作建议中仍需正常分析（不排除），但不作为新推荐标的

## 交易手续费计算规则

所有买卖建议的资金计算必须包含交易手续费，确保用户看到的是**真实交易成本**。

### 费率标准（2024年最新）

| 费项 | 费率 | 收取方向 | 最低收费 |
|------|------|---------|---------|
| 佣金 | 0.025%（万2.5） | 买入 + 卖出（双向） | 5元/笔 |
| 印花税 | 0.05% | 仅卖出（单向） | 无 |
| 过户费 | 0.001% | 买入 + 卖出（双向） | 无 |

### 计算公式

```python
def calc_buy_cost(price: float, quantity: int) -> dict:
    """计算买入总成本"""
    amount = price * quantity
    commission = max(amount * 0.00025, 5)  # 佣金，最低5元
    transfer_fee = amount * 0.00001        # 过户费
    total = amount + commission + transfer_fee
    return {
        "股票金额": round(amount, 2),
        "佣金": round(commission, 2),
        "过户费": round(transfer_fee, 2),
        "买入总成本": round(total, 2)
    }

def calc_sell_income(price: float, quantity: int) -> dict:
    """计算卖出实际到手"""
    amount = price * quantity
    commission = max(amount * 0.00025, 5)  # 佣金，最低5元
    stamp_tax = amount * 0.0005            # 印花税
    transfer_fee = amount * 0.00001        # 过户费
    net = amount - commission - stamp_tax - transfer_fee
    return {
        "股票金额": round(amount, 2),
        "佣金": round(commission, 2),
        "印花税": round(stamp_tax, 2),
        "过户费": round(transfer_fee, 2),
        "卖出实际到手": round(net, 2)
    }
```

### 输出要求

- **买入建议**：显示"所需资金 ≈ X,XXX元（含手续费约X元）"
- **卖出建议**：显示"预计到手 ≈ X,XXX元（扣除手续费+印花税约X元）"
- **盈亏计算**：盈亏 = 卖出到手 - 买入总成本（即扣除双向手续费+印花税后的真实盈亏）
- **止损/目标位**：计算盈亏百分比时也应考虑手续费影响（尤其是小额交易，手续费占比显著）

## 交易时段划分

| 时段 | 时间 | 分析重点 | 操作特点 |
|------|------|---------|---------|
| **开盘前** | 9:00-9:30 | 集合竞价、隔夜消息、外盘影响 | 预判开盘方向，制定当日计划 |
| **上午盘** | 9:30-11:30 | 开盘走势、热点板块、资金流向 | 捕捉早盘机会，快速反应 |
| **下午盘** | 13:00-15:00 | 上午延续性、尾盘动向、资金态度 | 确认趋势，调整仓位 |
| **收盘后** | 15:00后 | 全天复盘、板块表现、次日预判 | 总结当日，布局明日 |

## 执行流程

### 1. 识别当前时段

根据当前**上海时间（UTC+8）**自动判断：
- 9:00-9:30 → 开盘前分析
- 9:30-11:30 → 上午盘分析
- 13:00-15:00 → 下午盘分析
- 15:00后 → 收盘后分析

### 2.5 自动化候选股筛选（优先使用）

**优先使用程序化选股替代手动列举候选股代码：**

```python
from src.services.stock_screener import StockScreener

screener = StockScreener(manager)

# 从领涨板块自动筛选（按板块排行中的前3个板块分别筛选）
all_candidates = []
for sector in [领涨板块1名称, 领涨板块2名称, 领涨板块3名称]:
    results = screener.screen_from_sector(
        board_name=sector,
        top_n=5,
        min_score=60,
    )
    all_candidates.extend(results)

# 去重 + 按评分排序，取前10
seen = set()
unique = []
for c in sorted(all_candidates, key=lambda x: x["score"], reverse=True):

        seen.add(c["code"])
```

1. 板块成分股获取

3. 实时行情增强（PE/市值/换手率）
4. 30日K线获取 + 五维自动评分
5. 按评分降序排列

**降级策略**：若板块成分股API不可用（网络限流），降级为手动列举候选股代码 + 手动五维评分。

### 3. 候选股日K线技术分析（手动降级时使用）

**当自动化选股不可用时，手动执行此步骤。**

#### 3.1 获取日K线数据

```python
# 获取30日日K线（含均线和量比）
df, source = manager.get_daily_data(stock_code, days=30)
# 返回 DataFrame 字段: date, open, high, low, close, volume, amount, pct_chg, ma5, ma10, ma20, volume_ratio
```

#### 3.2 简化五维评分（满分100分）

对每只候选股，基于日K线数据执行以下评分：

**A. 趋势强度（30分）**

| 指标 | 评分标准 | 数据来源 |
|------|---------|---------|
| MA排列 | 完美多头(MA5>MA10>MA20>close上方): 12分, 部分多头: 7分, 粘合: 4分, 空头: 0分 | df['ma5'], df['ma10'], df['ma20'] |
| 均线方向 | MA5和MA10均上行: 10分, 仅MA5上行: 6分, 均下行: 0分 | 对比最近3日MA斜率 |
| 价格vs均线 | 现价>MA5>MA10>MA20: 8分, 现价>MA20: 5分, 现价<MA20: 0分 | 最新close vs 均线值 |

**B. 量价配合（25分）**

| 指标 | 评分标准 | 数据来源 |
|------|---------|---------|
| 近5日量价关系 | 上涨放量+下跌缩量: 12分, 上涨放量但下跌也放量: 6分, 上涨缩量: 2分 | df['volume'], df['pct_chg'] |
| 成交额趋势 | 近5日均额 > 近20日均额且递增: 8分, 持平: 4分, 萎缩: 0分 | df['amount'] 或 df['volume'] |
| 量比 | 最新量比>1.5: 5分, 1.0-1.5: 3分, <1.0: 0分 | df['volume_ratio'] |

**C. K线形态（20分）**

| 指标 | 评分标准 | 数据来源 |
|------|---------|---------|
| 近3日K线 | 连续阳线: 10分, 2阳1阴: 6分, 长下影线(探底回升): 4分, 连续阴线: 0分 | df['open'], df['close'] |
| 实体大小 | 近期阳线实体>2%: 5分, 1-2%: 3分, <1%: 1分 | (close-open)/open |
| 振幅趋势 | 振幅收窄后放大(突破): 5分, 正常: 3分, 持续放大(风险): 0分 | (high-low)/low |

**D. 乖离与空间（15分）**

| 指标 | 评分标准 | 数据来源 |
|------|---------|---------|
| 10日乖离率 | <8%: 5分, 8-15%: 3分, >15%: 0分, >20%: -3分 | (close-ma10)/ma10 |
| 距近30日低点 | <15%: 5分, 15-30%: 3分, >30%: 0分 | (close-min30)/min30 |
| 距近30日高点 | <5%: 5分(接近新高), 5-15%: 3分, >15%: 0分 | (max30-close)/max30 |

**E. 动量状态（10分）**

| 指标 | 评分标准 | 数据来源 |
|------|---------|---------|
| 近5日涨幅 | +3%~+10%: 5分, +10%~+20%: 3分, >+20%: 0分(超买), <0%: 2分(弱势) | 近5日累计pct_chg |
| 涨跌天数 | 近10日涨≥6天: 5分, 涨5天: 3分, 涨<5天: 0分 | 统计pct_chg>0的天数 |

#### 3.3 背离检测（扣分项）

| 背离类型 | 判定标准 | 扣分 |
|---------|---------|------|
| 量价背离 | 近3日价格创新高但成交量递减>15% | -8分 |
| 冲高回落 | 近3日最高价创新高但收盘价低于开盘价 | -5分 |
| 均线死叉 | MA5下穿MA10 | -5分 |

#### 3.4 筛选标准

```
最终得分 = 基础得分(A+B+C+D+E) - 背离扣分

≥75分：⭐⭐⭐⭐ 优先推荐（趋势向上概率大）
60-74分：⭐⭐⭐ 可推荐（有机会但需确认）
45-59分：⭐⭐ 不推荐（方向不明）
<45分：⭐ 规避（趋势偏空）
```

**只有得分≥60分的股票才进入最终推荐名单。**

### 4. 分时段分析策略

#### 📋 开盘前（9:00-9:30）

**分析重点**：
- 集合竞价情况（高开/低开/平开）
- 隔夜外盘表现（美股、港股）
- 重大消息面影响
- 昨日强势板块延续性

**推荐策略**：
- 关注集合竞价放量高开的板块龙头
- 避免低开跳水的弱势股
- 预判当日可能的热点方向
- **必须获取候选股30日日K线并评分≥60分**

**操作建议**：
- 制定当日交易计划
- 设置关注股票池
- 准备开盘后快速反应

---

#### 🌅 上午盘（9:30-11:30）

**分析重点**：
- 开盘30分钟走势（确认方向）
- 热点板块是否持续
- 资金流向和成交量
- 涨停板数量和质量

**推荐策略**：
- 优先推荐早盘放量上涨**且日K线趋势评分≥60分**的板块龙头
- 关注10:00-10:30的回调买点
- 避免冲高回落的弱势股
- **日K线显示均线空头排列的股票，即使当日涨幅大也不推荐**

**操作建议**：
- 超短线：可追强势股（评分≥75），严格止损
- 短线：等待回调确认后介入（评分≥60）
- 中线：观察为主，不急于操作

---

#### 🌆 下午盘（13:00-15:00）

**分析重点**：
- 上午走势的延续性
- 午后资金态度（加仓/减仓）
- 尾盘拉升或跳水信号
- 收盘前资金动向

**推荐策略**：
- 上午强势且午后继续放量**且评分≥60**：可追
- 上午强势但午后缩量：观望
- 午后异动拉升：谨慎，可能诱多
- **日K线显示乖离率>15%的股票不推荐，冲高回落风险大**

**操作建议**：
- 持仓者：根据走势决定是否加仓/减仓
- 空仓者：14:30后不建议新开仓
- 尾盘（14:45后）：除非明确信号，否则不操作

---

#### 🌙 收盘后（15:00后）

**分析重点**：
- 全天板块表现排名
- 涨停板复盘（封板质量）
- 资金流向统计
- 次日可能的热点延续

**推荐策略**：
- 推荐今日强势**且日K线趋势完好（评分≥60）**的板块
- 关注尾盘放量拉升的潜力股
- 避免尾盘跳水的弱势股
- **重点关注均线多头排列+量能配合的次日接力标的**

**操作建议**：
- 制定次日交易计划
- 设置次日关注股票池
- 总结当日得失

## 技能协作

- **data-fetching**：获取实时行情 + 日K线历史数据
- **stock-technical-analysis**：深度五维分析（完整版，可选，用于需要详细报告时）
- **portfolio-management**：结合持仓组合进行推荐

## 交易单位硬规则

**A股最小交易单位为100股（1手），所有买卖建议的数量必须为100的整数倍。**

### 严禁出现的描述

- ❌ "卖出50股" "买入150股" "减仓250股" — 非100整数倍
- ❌ "卖出半仓"（当持仓为100股时，半仓=50股，非法）
- ❌ "减仓30%" "减仓2/3"（当计算结果非100整数倍时，必须取整后再表述）

### 正确表述

- ✅ "买入100股" "卖出200股" "加仓300股"
- ✅ 持仓100股时：只能"持有"或"清仓（卖出100股）"，不存在"减仓"选项
- ✅ 持仓200股时：可"卖出100股"或"清仓（卖出200股）"
- ✅ 计算得到非整数：向下取整到100的倍数（买入方向少买，卖出方向少卖）
- ✅ 计算不足100股：不建议该操作（资金不足1手 / 持仓不支持减仓）

### ETF/基金单位

- ETF基金（如518800黄金基金）最小交易单位同样为100份
- 减仓/加仓数量必须为100份的整数倍

## 注意事项
- **推荐前必须获取30日日K线并完成五维评分，禁止仅凭当日涨幅推荐**
- **评分<60分的股票不得推荐，即使当日涨幅领先**
- **日K线均线空头排列（MA5<MA10<MA20）的股票原则上不推荐**
- **创业板（300/301开头）和ST股票一律不推荐，即使评分达标**
- **所有资金计算必须包含手续费（佣金万2.5+印花税0.05%+过户费0.001%），不得只算裸价×数量**
- 不同时段有不同的操作节奏
